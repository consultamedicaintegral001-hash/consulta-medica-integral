# Consulta Medica Integral

Sitio estatico gratuito optimizado para GitHub Pages, Netlify y Vercel.

## Archivos principales

- `index.html`: pagina principal con SEO, schema, redes sociales y agenda.
- `assets/css/styles.css`: estilos responsive.
- `assets/js/main.js`: animaciones, validacion del formulario, contador y botones flotantes.
- `robots.txt` y `sitemap.xml`: base SEO para publicacion.

## Publicacion gratuita

Sube el contenido de esta carpeta a GitHub Pages, Netlify o Vercel. Si usas un dominio distinto a `https://consultamedicaintegral.cl/`, actualiza el `canonical`, Open Graph, `robots.txt` y `sitemap.xml`.
